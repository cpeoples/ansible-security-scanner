# Scoring

Security scores are calculated based on issue severity:

| Severity | Points Deducted |
|----------|----------------|
| CRITICAL | -15 each |
| HIGH | -8 each |
| MEDIUM | -3 each |
| LOW | -1 each |

| Score Range | Rating |
|-------------|--------|
| 90-100 | Excellent |
| 75-89 | Good |
| 60-74 | Fair |
| 40-59 | Poor |
| 20-39 | Critical |
| 0-19 | Severe |
